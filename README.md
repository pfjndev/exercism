# 📂 Exercism Solutions

This repository serves as a personal backup and archive of my solutions to various programming exercises on the [Exercism](https://exercism.org) platform.

> **Exercism** is an online platform that provides code challenges (exercises) in many different programming languages, with the goal of helping people level up their programming skills through practice and mentorship.

## 🚀 Repository Structure

The exercises are organized by language track.

Each track folder contains subdirectories for individual exercises, including the solution files.

```text
Placeholder values which will be interpolated for each commit:

$track_slug:      The slug of the track               (e.g. "csharp").
$track_title:     The name of the track               (e.g. "C#")
$exercise_slug:   The slug of the exercise            (e.g. "hello-world")
$exercise_title:  The name of the exercise            (e.g. "Hello World")
$iteration_idx:   The iteration index of the exercise (e.g. "1")

.
├── [track_title]/             # e.g., javascript, python, rust
│   ├── [exercise_slug]/       # e.g., hello-world, two-fer
│   │   ├── [iteration_idx]    # e.g., 1, 2
│   │   │   ├── solution-file  # My implementation
│   │   │   └── test-file      # Optional test suite provided by Exercism
└── README.md

```
## 📖 How to Use This Repo

1. **Browse Solutions:** You can navigate through the folders to see how I solved specific problems.
2. **Run Tests:** If you have the [Exercism CLI](https://exercism.org/docs/building/tooling/cli) installed and the respective language environment set up, you can run tests for any exercise.

## ⚠️ Disclaimer

These solutions are for **reference and educational purposes only**.

* If you are currently working on these exercises, I highly recommend trying to solve them on your own first.
